На практической части пишем приложение, задача которого продемонстрировать студентам
принципы Чистой Архитектуры, а также познакомить студентов с дополнительными аннотациями Спринга.

Работаем пока без Мавена и Спринг-инициализатора, поэтому к уроку приложен
архив библиотек Спринга, которые вручную подключаем к проекту через интерфейс Идеи.
