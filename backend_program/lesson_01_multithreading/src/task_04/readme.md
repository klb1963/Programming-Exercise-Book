2. Демонстрируем, чем демон-поток отличается от обычного потока.

Обычные потоки: Обеспечивают выполнение основных задач программы и требуют завершения перед завершением работы JVM.

Демон-потоки: Обслуживающие потоки, которые не препятствуют завершению работы JVM и автоматически завершаются вместе с ней.
Использование демон-потоков полезно для фоновых задач, которые не должны блокировать завершение программы.
